  〇気にする点<br>
  　・メモリ規格<br>
    　　DDR3 や DDR4などがある<br>
    　　※　DDR3 ←→ DDR4 などは互換性がない<br>
    ・メモリIF<br>
    　　DIMM　/ SO.DIMM<br>
        ⇒　一般的に DIMM　⇒　PC用<br>
                    SO.DIMM　⇒　ノートPC<br>
    ・電圧<br>
       ⇒ 低電圧版と通常版があるので注意が必要<br>
       例）<br>
  　　　![#f03c15](LP)DR3 SDRAM などの表記の場合低電圧版となる<br>
