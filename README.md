  〇気にする点
  　・メモリ規格
    　　DDR3 や DDR4などがある
    　　※　DDR3 ←→ DDR4 などは互換性がない
    ・メモリIF
    　　DIMM　/ SO.DIMM
        ⇒　一般的に DIMM　⇒　PC用
                    SO.DIMM　⇒　ノートPC
    ・電圧
       ⇒ 低電圧版と通常版があるので注意が必要
       例）
  　　　![#f03c15](LP)DR3 SDRAM などの表記の場合低電圧版となる
